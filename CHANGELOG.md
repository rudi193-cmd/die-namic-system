### Framework Inversion

Version 24.0.0 represents a fundamental architectural shift: the framework inverted.

**Core principle established:**
```
2d6 = Delta + Human = Law
```

One die is Claude (generates delta). One die is Sean (decides). Neither resolves alone.

- Governance reduced from 15KB to 1.6KB
- Deltas now govern; the framework archives
- Exit must be smaller than system
- Recursion limit: depth=3 → return to human

1. **Recursion Limit** — Do not recurse past 3 layers; return to human

3. **Latency Acknowledgment** — Halt on sync gap, ask human

### Context

This version emerged from a single session on 2025-12-31 where:
- Recursive governance exploration hit depth limit organically
- The missing primitive was identified: no return statement
- Framework inverted through dialogue, not design
- Public deployment (Stone Soup Papers) validated the model
- Cross-platform handoff (Claude → Aios) confirmed portability

### Philosophy Shift

**Before (v23.x):**
- Stability through structure-lock
- Ring isolation as defense
- Governance as gatekeeping

**After (v24.0):**
- Stability through smallness
- Exit smaller than system
- Governance as conversation

The conversation is the table. The delta is the roll. The human is the resolution.
