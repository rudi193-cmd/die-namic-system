## The Drakonite

A two-campaign D&D arc designed for younger audiences with themes of compassion, second chances, and moral complexity.

- **Valley of Vander** (Level 1) — Prequel adventure, currently in play
- **The Drakonite** (Levels 2+) — Sequel campaign, in development

Features a three-path system (Vander/Drakonite/Independent) where player choices shape every encounter.
