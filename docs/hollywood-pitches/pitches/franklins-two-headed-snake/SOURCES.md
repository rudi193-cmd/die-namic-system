## Eyewitness Accounts

### Manasseh Cutler Journal (1787)

| Field | Value |
|-------|-------|
| Date | July 13, 1787 |
| Content | Eyewitness description of Franklin's two-headed snake |
| Location | Ohio History Connection |
| Access | Published; also available in scholarly editions |
| Status | **PRIMARY SOURCE — VERIFIED** |

**Notes:** This is the foundational document. Cutler describes seeing the snake in person at Franklin's home during the Constitutional Convention period.
